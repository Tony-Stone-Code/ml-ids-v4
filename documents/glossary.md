# Simple Glossary

- Accuracy: percent of correct predictions.
- Precision: when the model predicts an attack, how often it is right.
- Recall: how many real attacks the model finds.
- F1 score: balance between precision and recall.
- Training data: data used to teach the model.
- Testing data: data used to evaluate the model.
- Class imbalance: when some attack types are much rarer than others.
- Confusion matrix: a table that shows correct and incorrect predictions.
- Model: the trained program that makes predictions.
- Feature: an input column used for prediction.
